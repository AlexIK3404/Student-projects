# Student Projects Portfolio

Небольшой портфолио‑репозиторий с учебными работами по Python и C++.  
This repository collects учебные проекты and лабораторные работы for a student portfolio.

## Структура / Structure

| Путь / Path | Описание / Description |
| --- | --- |
| `python/` | Аналитика данных, презентации и отчёты по Python‑курсу. |
| `python/algorithms-and-data-structures/` | Алгоритмы и структуры данных (AnSD): лабораторные и курсовая. |
| `C++/` | Лабораторные и курсовая работа по дисциплине «Программирование». |

> Примечание: папка `C++` содержит материалы по C++ и дисциплине «Программирование».

## Как пользоваться / How to use

- Для просмотра материалов просто открывайте PDF/PPTX/DOCX файлы.
- Для Jupyter‑ноутбуков используйте JupyterLab/VS Code.
